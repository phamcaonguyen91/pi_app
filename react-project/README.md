# React project generated from uploaded archive

This project uses **Vite + React**. Files you uploaded have been copied into `src/` where possible.

## Quick start

1. Install dependencies:
```bash
npm install
# or
yarn
# or
pnpm install
```

2. Run dev server:
```bash
npm run dev
```

3. Build:
```bash
npm run build
```

Files from your uploaded ZIP were placed under `src/`. If any uploaded files conflicted with scaffold files they were suffixed with `.uploaded`.
